    1. What is Semantic HTML? 
	Semantic HTML is HTML that has a specific meaning, such as p tags, h1-h6, header, footer, etc.. 
	It is basically everything except divs and spans.

    2. What is HTML used for? 
	HTML is used as the foundation to create a website. It tells the browser how to display information.

    3. What is an attribute and where do we put it? 
	Attributes are a way to give more information to a tag. Attributes are inside the opening tag, after the initial
	tag label.

    4. What is the h1 tag used for? How many times should I use it on a page?
	The h1 tag is used for the title of the page. It should only be used once.

    5. Name two tags that have required attributes
	<img src> (<--'src' is the attribute) <a href> (<--'href' is the attribute)

    6. What do we put in the head of our HTML document? 
	The head contains information that is not displayed on the page such as imported fonts, language information, etc.
	
    7. What is an id? 
	An id is an attribute that labels your element with a unique identifier for styling in CSS or javascript. 

    8. What elements can I add an id to? 
	Any element can have an id. 

    9. How many times can I use the same id on a page? 
	You should only use a specific id once.

    10. What is a class? 
	A class is an attribute similar to an id, but is used for styling 1 or more elements.

    11. What elements can I add a class to? 
	Any element can have a class. 

    12. How many times can I use the same class on a page? 
	As many times as you need.

    13. How do I get my link to open in a new tab?
	target="_blank"

    14. What is the alt attribute in the image tag used for? 
	For screen readers to describe a photo for users with disabilities.

    15. How do I reference an id?
	#id

    16. What is the difference between a section and a div?
	Divs are non-semantic. 

    17. What is CSS used for? 
	CSS is used to style web pages.

    18. How to we select an element? Example - every h2 on the page
	h2 {}

    19. What is the difference between a class and an id? - Give me an example of when I might use each one
	A class can be used multiple times, an id is for one specific thing. For example if you have multiple <p> tags, you 
	should assign it a class. But something like a header or a footer can use an id because there is only one of them. 

    20. How do we select classes in CSS?
	.className
	
    21. How do we select a p element with a single class of “human””?
	.human

    22. What is a parent child selector? When would this be useful?
	A parent child selector is a way to grab a specific element within an element, such as an anchor inside of a footer.
	Ex: nav a {}
 
    23. How do you select all links within a div with the class of sidebar?
	.sidebar a {}

    24. What is a pseudo selector?
	A pseudo selector is for a special action on the page, such as mouse over, click, etc.

    25. What do we use the change the spacing between lines?
	line-height

    26. What do we use to change the spacing between letters?
	letter-spacing

    27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?

    30. How do I write a comment in CSS?

Stretch Questions

    1. What is the difference between an inline element and a block element?
	A block level element is an element that takes up its own line on a web page. An inline element does not take up its 
	own line and can have other elements on the same line. 

    2. What happens when an element is positioned absolutely? 
	Absolute positions attach themselves to the closest ancestor element. 

    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
	display: inline-block

    4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default
	Display block by default: div, h1, p 
	Display inline by default: span, a
	Display inline-block by default: img

    5. In your own words, explain the box model. What is the "fix" for the box model, in other words, how do we make all elements respect the width we've given them?
	The box model is box that contains different pieces that cascade outwards when styling, starting with content, then
	padding, border, and margin. 
	You can fix the issues with the box model using border-box, which includes the border and padding within the width you assign.
